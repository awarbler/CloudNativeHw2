# AJW4987_HTML_Site


React Homework (ECE 382V)

This repository contains the two required artifacts for the React Homework assignment:

- Task 1: A static single-page website built with HTML + CSS and deployed with GitHub Pages.
- Task 3: A frontend-only React application implementing a Projects page (reusable components, props, and state).

Scope note: No backend, API, authentication, or database is included for this homework.

## Task 1 (Static Website)
Location:
- index.html (repo root)
- stylesheet.css (repo root)
- images/ (repo root)

Rubric features implemented:
- External stylesheet
- Meta description tag
- At least two <div> content sections
- At least one external link and hover styling
- Tag selector + class selector (used 2+ times) + id selector
- Box model styling (margin/padding/border)
- Safe-for-work image with alt text

## GitHub Pages Deployment
This repo is deployed using GitHub Pages from the `main` branch and `/ (root)` folder.

To view the deployed site:
- Go to Settings → Pages in this repo to find the published URL.
